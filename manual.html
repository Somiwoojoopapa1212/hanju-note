앱 소개 및 주요 기능

주향 노트(酒香ノート)는 일본주와 한국전통주 애호가를 위한 시음 기록 앱입니다. 별도의 회원가입 없이 설치하여 바로 사용할 수 있으며, 모든 데이터는 기기에 안전하게 저장됩니다.

📝
시음 노트 (메인)
한국전통주 8종 + 일본주 8종 + 기타 직접입력. 향·맛·여운 텍스트 및 점수 기록.
🎯
향미 프로필
달콤·산미·감칠맛·곡물쌀·과일향·꽃향 6축 레이더 차트. 사케·전통주 특화.
🖼️
인스타 공유 카드
1080×1080px PNG 자동 생성. 술 정보·향미 차트·노트 포함.
📊
통계 분석
카테고리·종류·지역별 분포. 취향 분석 리포트 및 다른 사람 통계.
🔖
위시리스트
마셔보고 싶은 술 저장. 우선순위(🔴🟡🔵) 설정, 구매완료 처리.
💾
데이터 백업
전체 데이터(사진 포함) JSON 파일로 내보내기·가져오기.
💡 오프라인 사용 가능: 앱 설치 후에는 인터넷 없이도 시음 노트 조회 및 작성이 가능합니다. 다른 사람 통계는 인터넷 연결 시에만 작동합니다.

설치 방법

주향 노트는 PWA(Progressive Web App) 방식으로 배포됩니다. 앱 스토어 설치 없이 브라우저에서 직접 설치합니다.

https://somiwoojoopapa1212.github.io/hanju-note/

Android 안드로이드 설치

  • Chrome 브라우저로 위 주소에 접속합니다.
  • 하단에 앱 설치 배너가 나타나면 탭하거나, ⋮ 메뉴 → 홈 화면에 추가를 선택합니다.
  • 홈 화면에 주향 노트 🍶 아이콘이 추가됩니다.

iOS 아이폰 / 아이패드 설치

⚠️ 주의: iOS에서는 반드시 Safari 브라우저로 접속해야 합니다.
  • Safari로 위 주소에 접속합니다.
  • 하단 공유 버튼(⬆️) 탭 → 홈 화면에 추가를 선택합니다.
  • 홈 화면에 주향 노트 🍶 아이콘이 추가됩니다.

종류 선택 — 기타 직접 입력

종류 드롭다운에서 기타 (직접 입력)을 선택하면 자유 텍스트 입력 필드가 나타납니다. 목록에 없는 술(이화주, 삼해주, 안동소주 등)을 직접 입력하세요.

📷 사진 등록 (정사각형 크롭)

사진 존을 탭하여 갤러리에서 이미지를 선택하면 정사각형 크롭 화면이 나타납니다.

  • 드래그로 위치 조정
  • 슬라이더로 확대/축소
  • 적용으로 저장

🎯 향미 프로필 (Flavor Profile)

6개의 슬라이더(각 0~5단계)를 조작하면 아래에 레이더 차트 미리보기가 실시간으로 업데이트됩니다. 슬라이더는 전체 너비로 표시되어 모바일에서도 편리합니다.

🍋

🌊
감칠맛 (Umami)
깊은 감칠맛·발효향·효모
🌾
곡물·쌀 (Grain)
쌀·보리·누룩·곡물향
🍊

🌸

💡 팁: 향미 데이터가 많을수록 취향 분석 리포트가 정확해집니다. 시음마다 입력해 보세요.

자주 묻는 질문 (FAQ)
Q. 목록에 없는 술 종류도 기록할 수 있나요?

A. 네. 종류 선택에서 기타 (직접 입력)을 고르면 자유롭게 입력 가능합니다. 이화주·삼해주·안동소주 등 어떤 술도 기록할 수 있습니다.

Q. 정미율(精米步合)이란 무엇인가요?

A. 쌀을 도정한 비율로, 숫자가 낮을수록 많이 깎아낸(고급) 사케입니다. 예: 60%는 쌀 40%를 깎아낸 것, 23%는 다이긴조 최고급 등급.

Q. 기기를 교체할 때 데이터를 옮길 수 있나요?

A. 네. 기존 기기에서 데이터 내보내기로 JSON 저장 후, 새 기기에서 데이터 가져오기 → 덮어쓰기로 이전하세요.

Q. 향미 프로필을 입력하지 않아도 되나요?

A. 네, 선택 항목입니다. 단, 향미 데이터가 많을수록 취향 분석 리포트가 정확해집니다.

Q. 피드백이나 버그는 어디에 제보하나요?

A. 더보기(☰) → 피드백하기에서 이메일로 제보하실 수 있습니다.
