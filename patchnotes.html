v 1.0 주향 노트 출시 🎉
✨ 신기능 시음 노트 (메인 기능) — 한국전통주 8종 + 일본주 8종 + 기타 직접입력. 종류 선택 시 '기타'를 고르면 자유롭게 텍스트 입력 가능.
✨ 신기능 향미 프로필 레이더 차트 — 사케·전통주 특화 6축: 달콤 · 산미 · 감칠맛 · 곡물·쌀 · 과일향 · 꽃향. 슬라이더가 전체 너비를 사용해 모바일에서도 편리하게 조작.
✨ 신기능 사케·전통주 특화 입력 필드 — 정미율(精米步合) · 원료(米品種) · 음용 온도(冷/常温/ぬる燗/熱燗) · 투명도(맑음~불투명) · 액체 색깔(무색투명~탁백색) 지원.
✨ 신기능 한국·일본 지역 분리 선택 — 한국(서울·경기/전라/충청/경상/강원/제주) + 일본(니가타/교토/효고/아키타/야마가타/이시카와) 지역 옵션.
✨ 신기능 향 · 맛 · 여운 점수 슬라이더 (각 0~100) 및 종합 점수. 점수를 입력하지 않을 경우 × 버튼으로 미설정 상태로 초기화 가능.
✨ 신기능 인스타그램 공유 카드 — 1080×1080px PNG 자동 생성. 술 이름 · 향미 레이더 차트 · 시음 노트 · 날짜 포함.
✨ 신기능 통계 분석 — 카테고리별(한국/일본) · 종류별 · 지역별 · 점수 분포. 향미 데이터 기반 취향 분석 리포트 (스타일 추천 포함).
✨ 신기능 위시리스트 — 마셔보고 싶은 술 저장. 우선순위(🔴🟡🔵) · 목표 가격 · 종류 설정. 완료 처리로 구분 관리.
✨ 신기능 데이터 내보내기 / 가져오기 — 전체 데이터(사진 포함)를 JSON 파일로 백업 및 복원. 병합 · 덮어쓰기 모드 지원.
✨ 신기능 PWA 지원 — 홈 화면에 앱 설치 가능 (백자 잔 아이콘). 오프라인에서도 시음 노트 조회 · 작성 가능.
🎨 디자인 사케 특화 디자인 — 맑고 청량한 쿨 블루 색조 (primary: #4a8bc4). 백자 잔에 불투명한 막걸리·니고리 스타일 액체가 담긴 SVG 스플래시 아이콘.
